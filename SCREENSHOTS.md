## ✨ Key Features Shown

1. **Sustainable Mission** - Clear messaging about reducing textile waste
2. **Fair Coin System** - 120 free coins to start, 20 coins per swap
3. **Easy Process** - Upload clothes, browse marketplace, negotiate, and meet
4. **Smart Chat** - Quick reply buttons for efficient communication
5. **OTP Verification** - Secure in-person swap confirmation
6. **Status Tracking** - Visual progress from negotiating to completed
7. **User Profiles** - Digital wardrobe with item management
8. **Real-time Updates** - Live message notifications and status changes

## 🎨 Design Highlights

- Accessible color contrast and UI elements

## 📱 User Flow

1. **Sign Up** → Get 120 free coins
2. **Upload Items** → Add clothes to digital wardrobe
3. **Browse Marketplace** → Find items you want
4. **Request Swap** → Start conversation (costs 20 coins)
5. **Negotiate** → Use quick replies to communicate
6. **Schedule Meeting** → Agree on time and place
7. **Generate OTP** → Both users get verification codes
8. **Meet & Swap** → Verify OTPs and complete exchange
9. **Success!** → Items swapped, both users happy
